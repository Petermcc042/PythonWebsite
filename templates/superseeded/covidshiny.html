Covid R Shiny Dashboard Project

UK Gov Covid Data visualisations Project

Peter McClintock

25/11/2021

Overview

  • This project was testing out basic visualisations using R and Shiny. I wanted to challenge myself to begin creating end visualisations for my data analysis projects
  • The language used for the project was solely R which only required additional learning of the reactive programming necessary for Shiny

Data

The data was sourced from the UK Covid Data website which gives an in depth breakdown of the daily Coronavirus figures from the begining of the pandemic. The data I worked with was downloaded as of the 23/11/2021.

As I was focused on producing the visualisations and learning Shiny I did not commit to a deep dive into the data picking only the main fields of the four downloaded datasets. Each of the datasets used can also be downloaded from my github, along with the code used to create the app.

Visualisations

  • Firstly, I wanted to create an interactive plot of the four main variables, number of cases, deaths, vaccinations, and hospital admissions per day. For this I merged the four variables into one dataframe based on the longest time frame. As such the vaccination level at start comes in as NA values. From this I used the plotly package due to its benefits of easy interactivity. For styling I added a second axis, advanced period selection features and colour.
  • The second part of the dashboard is a closer look each series and is calculated as the input changes selection. The first plot is a simple boxplot made with ggplot. Secondly, to help add to the box plot is a simple dataframe of summary statistics. The final plot is a histogram showing the distribution of the selected data. With it I included a numeric slider to control the number of bins.

Shiny

Working with Shiny proved more difficult than I would have initially hoped coming from a well established background in R. This is due to the use of reactive data which requires additional learning in order to function. However, even with this basic first dive into Shiny I believe I have managed to produce a professional looking dashboard with some basic information. As such I have hope for future projects and what they can yield.
