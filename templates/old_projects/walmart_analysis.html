  • The project is a basic dive into some data from Walmart stores
  • I finish the project by using a random forest to predict weekly sales for 4 stores and show the predictions.

Data

The data comes from Kaggle and is a simple dataset containing 8 columns of data with 45 stores. It contains the weekly sales data for these stores from 2010 to 2012 along with the corresponding features during that week. Within the data are the following fields.

  • Store - Integer representing store
  • Date - The date of the sales week
  • Weekly_Sales - the sales for the given store
  • Holiday_Flag - Whether the week is a special holiday week (1) or not (0)
  • Temperature - Not specified as to whether it is average or not
  • Fuel_Price - Cost of fuel in the region (again not specified if it as an average over the week)
  • CPI - the prevailing consumer price index value
  • Unemployment - the prevailing unemployment rate

The holiday events included are the Superbowl, Labour day, Thanksgiving and Christmas.

Predicting data using the CARET Package

The caret package in r stands for Classification and Regression Training and contains the main functions required for creating predictive models. To begin the model building process I split each of the store’s data in an 80:20 split. This is a random split that occurs and leaves the training data split with 5130 observations and the test data with 1305 observations.

To help with the problem of overfitting 10-fold cross validation is used. The caret package allows us to call one function and pass in this as a parameter just before run time. The accuracy measure that will be used is Root Mean Squared Error as predicting weekly sales is a regression problem. For the model I have chosen a random forrest and in keeping with the documentation we set mtry and ntrees as default.

  • mtry: Number of variables randomly sampled as candidates at each split.
  • ntree: Number of trees to grow.

Below is the code used to run the model.

seed <- 1
  
  data_train = data.frame()
  data_test = data.frame()
  
  for (i in 1:45) {
    temp <- data %>% filter(Store == i) %>% select(Store, Holiday_Flag, Temperature, Fuel_Price, CPI, Unemployment, Weekly_Sales)
    trainIndex <- sample(1:length(temp$Store), (0.8*length(temp$Store)))
    data_train <- rbind(data_train, temp[ trainIndex,])
    data_test <- rbind(data_test, temp[-trainIndex,])
  }
  
  control <- trainControl(method="cv", number=10)
  metric <- "RMSE"
  mtry <- sqrt(ncol(data_train[,1:length(data_train)-1]))
  
  set.seed(seed)
  
  tunegrid <- expand.grid(.mtry=mtry)
  rf_default <- train(Weekly_Sales~., data=data_train, method="rf", metric=metric, tuneGrid=tunegrid, trControl=control)
  print(rf_default)

predictions <- predict(rf_default, newdata = data_test)
  data_test <- cbind(data_test, predictions)
  x = round(sqrt(mean((data_test$Weekly_Sales - predictions)^2)),2)
